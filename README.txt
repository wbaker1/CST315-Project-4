README for CST-305 Project 4 by Wesley Baker and Matthew Powers

In order to run this in python, one must do the following:

1) Have python installed

2) Download and install PiP if not already installed (comes with most python versions)

3) Use the following commands to install numpy, and matplotlib
    pip install numpy
    pip install matplotlib

4) Finally just download and install CST305P4.py and run it by opening command prompt, entering cd <directory where the CST305P4.py (should be Downloads)>

5) Once in the directory where the file is located run the following command: python CST305P4.py, give it a second to generate the graph it should appear after 1-2 seconds.

6) Finally, just close each graph to have the next one appear
